Before Getting into design aspects, I'll go through all the tags and elements to look after in HTML.

<html>:

This is the first tag which wraps all the web page. This consists of two major tags. Head and Body. Which classifies the data in the web page for their specific functions.

1. <head>:

Head element is a container for metadata (data about data) and is placed between the html tag and the body tag. Metadata is data about the HTML document. Metadata typically define the document title, character set, styles, scripts, and other meta information. Head tag contatins title tag which is the name displayed in the new tab when the page loads. And other tags like base,link,meta,style,script etc...

Tags placed inside <head>:

Tag Purpose
<base> Specifies a default URL and target for all links on a page
<style> Adds style information to a page(CSS).
<link> Links to an external style sheet
<title> Represents the title of the overall HTML document
<meta> Meta tag defines metadata about an HTML document.Metadata is used by browsers, search engines, and other web services.
<script> Script tag is used to embed a client-side script (JavaScript).

2. <body>:

Body tag contains the data which is displayed in the browser window, which has the text, hyperlinks to other pages etc. And other tags used inside body tag are h1,h2,h3,p,span,article etc...

Tags placed inside <body>:

Tag Purpose
<a> Defines a hyperlink, which is used to link from one page to another.
<h1> <h2> <h3> <h4> <h5> <h6> To define HTML headings, Subtitles of some content.
<article> Specifies independent, self-contained content.
<blockquote> Specifies a section that is quoted from another source.
<br> Inserts a single line break.
<button> Defines a clickable button.
<b> Makes text bold enclosed by the tag.
<data> This element provides both a machine-readable value for data processors, and a human-readable value for rendering in a browser.
<datalist> Specifies a list of pre-defined options for an input element.tag is used to provide an "autocomplete" feature for input elements. Users will see a drop-down list of pre-defined options as they input data.
<del> Strike a line through text enclosed within the tag.
<form> Used to create an HTML form for user input. It also contains one or more elements such as input, textarea, button, option, select, label etc
<img> Used to embed an image in an HTML page.
<picture> More flexibility while specifying image resources. Working with multiple images.
<input> Most important form element. Specifies an input field where the user can enter data.
<main> Contents inside main tag should be unique, which is not repeated across documents such as sidebars, header, footer etc.
<nav> Defines a set of navigation links.
<ol> Ordered list of data includes alphabetical or numerical.
<ul> Unordered list of data with bullet points.
<p> Defines a single paragraph. Browsers automatically add single blank line before and after each p element.
<output> Used to display output after some arithmetic computation.
<progress> Represents the completion progress of a task. Can be used in conjunction with JavaScript to display the progress of a task.
<s> Text with a strike in the middle. To display no longer correct, accurate or relevant data.
<section> We can define sections in a document and design accordingly.
<source> Used to specify multiple media sources for media elements such as audio, video, picture etc.
<span> It is a inline container used to markup a part of a text. Manipulated with JS or styled with CSS using the clas or id attribute.
<div> Defines a secion or a section of a document. The div tag is styled using the class or id attribute. By default browsers insert line break before and after div element.
<sub> To display subscript to a data.
<sup> To display superscript to a data.
<details> Used to display data only when the summary of the details is clicked.
<summary> Used as a heading for details tag, which then clicked displays data associated with it.
<table> Construct table in HTML. It also consists of other tags such as thead, tbody, tr, th, td etc.

The above tags represented the previous version of HTML5. Current version supports other tags which are useful in multimedia applications.

The tags associated with HTML5 are: audio,video(embbed audio and video file in a page),svg(vector graphics),header, footer(explicit tags to classify them without using the div tags), nav, progress, placeholder in place of input field, email attribute in input, database storage SQL databases, article, section, meter etc.
